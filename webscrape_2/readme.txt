this projects downloads the selected books from website royalroad.com
the main reason for making this project is to avoid the stubs of books
there is also a project that downloads the entire site

HOW TO
1. run insert_into.py and enter the full link then type rr for website name
2. run create.py
3. run get_content.py this will start the download of the book or books depending on how many new links there are

this project will also update any books with the new chapters
